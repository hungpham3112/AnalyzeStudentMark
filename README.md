# AnalyzeStudentMark

## Prerequisite

Python >= 3.6

numpy >= 3.4

## Installation

Clone the repository to your local machine and run main.py:

```pwsh
git clone https://github.com/hungpham3112/AnalyzeStudentMark.git 
cd AnalyzeStudentMark
python -m pip install -r requirements.txt
python main.py
```
